🌐 Complete HTML Tags List – Grouped Format (HTML5) 🏗️ 1. Basic Structure Tags Declares the document is HTML5 Root of the HTML document Contains metadata Page content Sets the browser title <meta> Metadata (charset, viewport) <link> Links external files (like CSS) <style> Internal CSS styling <script> JavaScript (internal or external) <base> Sets base URL for links 📝 2. Text Formatting & Content <h1> to <h6> Headings <p> Paragraph <br> Line break <hr> Horizontal line <pre> Preformatted text <blockquote> Block quote <q> Inline quote <abbr> Abbreviation <cite> Citation <code> Code snippet <kbd> Keyboard input <mark> Highlight text <small> Smaller text <strong> Bold <em> Italic <del> Strikethrough <ins> Underline (inserted) <sub> / <sup> Subscript / Superscript <b> / <i> / <u> Bold / Italic / Underline (non-semantic) 🔗 3. Hyperlinks & Anchors <a href="..."> Hyperlink target="_blank" Open in new tab download Trigger file download 🖼️ 4. Images & Media <img> Embeds image <figure> Wraps image/media <figcaption> Caption for figure <audio> Embeds audio <video> Embeds video <source> Media source (audio/video) <track> Subtitles or captions <embed> Embeds external content (PDF, Flash) <object> Embeds multimedia (like PDF, SVG) <iframe> Embeds external web pages (e.g. YouTube) 📋 5. Lists <ul> Unordered list <ol> Ordered list <li> List item <dl> Description list <dt> Term <dd> Definition 🧾 6. Forms & User Input <form> Form container <input> User input (text, number, etc.) <label> Label for input <textarea> Multiline input <select> Dropdown <option> Option inside dropdown <optgroup> Group related options <fieldset> Group fields <legend> Title for fieldset <button> Clickable button <datalist> Predefined input values <output> Result of calculation <progress> Shows task progress <meter> Shows scalar measurement 🔘 Common <input> types: text, password, email, number, radio, checkbox, submit, reset, date, file, range, search, tel, url, color, hidden 📦 7. Containers & Grouping <div> Block container <span> Inline container <template> Invisible content used with JavaScript <slot> Placeholder for content (Web Components) 🧠 8. Semantic HTML5 Tags <header> Top of page/section <footer> Bottom of page/section <nav> Navigation links <main> Main content area <section> A section of content <article> Independent content (blog post, comment) <aside> Sidebar content <figure> Groups media <figcaption> Caption for media <time> Represents a date/time 📊 9. Tables <table> Table element <tr> Table row <td> Table data (cell) <th> Table heading <thead> / <tbody> / <tfoot> Groups rows <caption> Table title colspan, rowspan Merge cells 🌍 10. Miscellaneous <noscript> Content shown if JavaScript is disabled <canvas> Drawing area using JS <svg> Vector graphics <math> MathML for equations <details> Expandable area <summary> Visible heading for <details> <dialog> Popup dialog <menu> Context menu <menuitem> Item in a menu 🧼 11. Common Attributes Attribute Purpose id Unique identifier class Reusable CSS styling style Inline CSS title Tooltip text alt Alternative text for images src Source (image/video/script) href Link target (for <a> or <link>) target Link behavior (_blank, _self) type Type of input or script value Default value (inputs) name Name of form control disabled, readonly, required, checked Input states
