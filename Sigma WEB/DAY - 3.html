✅ HTML Tags Grouped by Category – Beginner Friendly 🏗️ 1. Structural Tags Declares the document type Root element of HTML Contains meta information Holds the content of the web page Sets the page title <meta> Provides metadata <link> Links external resources (CSS) <script> Links JavaScript code 📝 2. Text Content & Formatting <h1> to <h6> Headings (from largest to smallest) <p> Paragraph <br> Line break <hr> Horizontal line <strong> Bold text <em> Italic text <mark> Highlights text <small> Smaller text <del> Deleted (strikethrough) <ins> Inserted (underlined) <blockquote> Quoted block <pre> Preformatted text <code> Displays code 🔗 3. Links & Anchors <a href="url"> Hyperlink target="_blank" Opens link in a new tab download Download linked file 🖼️ 4. Images & Media <img src="..." alt="..."> Embeds image <figure> Wraps image & caption <figcaption> Caption for image <audio> Adds audio <video> Adds video <source> Defines media file <iframe> Embeds external page or video 📋 5. Lists <ul> Unordered list (bullets) <ol> Ordered list (numbers) <li> List item <dl> Description list <dt> Term in a description list <dd> Definition of the term 🧾 6. Form & Input <form> Form element <input> User input (text, checkbox, radio, etc.) <textarea> Multi-line input <label> Label for input <button> Clickable button <select> Dropdown <option> Option inside dropdown <fieldset> Groups form inputs <legend> Caption for fieldset 📦 7. Layout / Container Tags <div> Generic container (block-level) <span> Generic container (inline) <section> A section of content <article> Independent article <header> Top part of a page <footer> Bottom part of a page <nav> Navigation links <main> Main content of the page <aside> Sidebar content 📊 8. Tables <table> Table element <tr> Table row <td> Table data (cell) <th> Table header <thead> Table header group <tbody> Table body group <tfoot> Table footer group colspan / rowspan Merge cells 🧠 9. Meta & Head Utility Tags <meta charset="UTF-8"> Character encoding <meta name="viewport"> Responsive behavior <title> Browser tab title <link rel="stylesheet"> Link CSS <script> Link JavaScript 📌 10. Semantic HTML5 Tags (Recommended for SEO & Accessibility) <header> Page or section header <footer> Page or section footer <main> Main content <nav> Navigation <article> Self-contained content <section> Thematic grouping <aside> Side info (ads, tips) <time> Date/time
