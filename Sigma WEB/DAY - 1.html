📘 Day 1: Introduction to HTML (HyperText Markup Language) ✅ What is HTML? HTML stands for HyperText Markup Language. It's the standard language used to create web pages. HTML structures content using tags. 📌 Basic Structure of an HTML Document: My First Web Page

Hello, World!

This is my first webpage.

🔍 Explanation of the Structure: Declares the document type (HTML5). Root of the HTML document. Contains meta info (title, CSS, JS, etc.). Title shown in the browser tab. <body> Content visible on the page. 🔤 Common HTML Tags: <h1> to <h6> Headings (<h1> is largest) <p> Paragraph <a href=""> Anchor (Link) <img src="url" alt="desc"> Image <br> Line break <hr> Horizontal line <strong> Bold <em> Italic <ul> / <ol> Unordered / Ordered list <li> List item 🔗 Example: Link and Image <a href="https://google.com">Go to Google</a> <img src="logo.png" alt="Site Logo"> 🧠 Tips: Always close your tags (<p>...</p>). Use indentation for better readability. Save your file with .html extension (e.g., index.html). Use a browser like Chrome to open and test your HTML files. ✅ Practice Task for Day 1: Create a webpage with: A heading A short paragraph about yourself A link to your favorite website An image (can use any online image URL) A horizontal line and a list of your hobbies
